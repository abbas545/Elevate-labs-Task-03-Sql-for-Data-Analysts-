# Task 3 - SQL for Data Analysis

##  Objective
The objective of this task is to use SQL queries to extract and analyze data from a database.

##  Tools Used
- SQLite
- DB Browser for SQLite

##  Dataset
Sample Ecommerce dataset created using SQL tables (Customers and Orders).

##  Concepts Covered
- SELECT Statement
- WHERE Clause
- ORDER BY
- GROUP BY
- HAVING Clause
- JOINS (INNER JOIN, LEFT JOIN)
- Subqueries
- Aggregate Functions (SUM, AVG)
- Views
- Indexing
- NULL Value Handling

##  Outcome
Learned how to:
- Query structured data
- Perform data analysis using SQL
- Create views for analysis
- Optimize queries using index

##  Files Included
- task3.sql → All SQL queries
- Screenshots → Query outputs
